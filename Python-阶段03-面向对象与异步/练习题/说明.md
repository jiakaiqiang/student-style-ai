# 阶段 3 动手练习（写 → 跑 → 审 → 讲）

> 这组练习按「你写、AI 审」的循环来，目的是把阶段 3 的概念变成手感。
> 做完这组再去答 `python-stage-03-assessment.md`。

## 规则（很重要）

1. **关掉 AI 补全**，全程手敲。写错是学习信号，不是事故。
2. 每个文件写完后**自己先跑一遍**：`python 文件名.py`
3. 报错不要直接问我要答案，先自己读 Traceback 最后一行，改一次；改不动再来问「为什么」。
4. 带 `# 预测：` 的地方，**先写下你预测的输出，再运行验证**。预测对了才说明心智模型准了。
5. 全部做完后告诉我，我来逐个 review，然后进入「你讲给我听」环节。

## 老规矩

```text
顶层代码：0 个空格
函数/类/方法体：4 个空格
同一代码块：缩进必须一致
```

字段名、文件名、输出格式：**严格按题目要求**，一个字都不要差。
