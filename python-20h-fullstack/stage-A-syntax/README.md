# 阶段 A：语法翻译（6 小时）

核心目标：**能流畅地用 Python 写出你已经会用 JS 写的东西。**

## 重点知识点

| 优先级 | 内容 | 对应你已会的 JS 概念 | 重点差异 |
|---|---|---|---|
| ★★★ | 缩进与代码块 | `{}` 大括号 | Python 用**缩进**代替 `{}`，这是你最大的坑 |
| ★★★ | 变量与基本类型 | `let/const` | 不用声明关键字，直接 `x = 1` |
| ★★★ | 列表 / 字典 | 数组 / 对象 | `list` ≈ Array，`dict` ≈ Object |
| ★★★ | 函数 `def` | `function` | 默认参数、关键字参数比 JS 强大 |
| ★★★ | 条件 / 循环 | `if / for` | `for x in list:` 是默认写法，没有 C 式 for |
| ★★ | f-string 字符串 | 模板字符串 `` `${}` `` | `f"hello {name}"` |
| ★★ | 列表推导式 | `.map() / .filter()` | `[x*2 for x in nums]` 一定要练熟 |

### 题 2：列表基本操作

**文件**：`exercise_02_list.py`

给定一个列表：

```python
fruits = ["apple", "banana", "cherry"]
```

完成三件事：
1. 在末尾加一个 `"orange"`
2. 打印列表的长度
3. 用 `for` 循环逐行打印每个水果

### 题 3：字典 + 循环（重点）

**文件**：`exercise_03_dict.py`

给定一个字典，表示三个人的年龄：

```python
ages = {"小明": 18, "小红": 22, "小刚": 30}
```

要求：
1. 打印小红的年龄
2. 用 `for` 循环，按 `名字: 年龄` 的格式逐行打印所有人
3. 把小明的年龄改成 19，再打印小明的年龄

> 提示：遍历字典用 `for key, value in ages.items():`

### 题 4：函数 def（重点）

**文件**：`exercise_04_function.py`

写一个函数 `add(a, b)`，返回两数之和。 
然后调用它，打印 `add(3, 5)` 的结果。

进阶：给 `b` 一个**默认值** `10`，这样 `add(3)` 也能跑，结果应该是 `13`。

### 题 5：列表推导式（前端的 .map / .filter）

**文件**：`exercise_05_comprehension.py`

给定：

```python
nums = [1, 2, 3, 4, 5, 6]
```

用**列表推导式**（一行）完成：
1. 得到每个数的平方：`[1, 4, 9, 16, 25, 36]`
2. 只保留偶数：`[2, 4, 6]`

> 提示： 
> 平方 → `[x*x for x in nums]` 
> 过滤 → `[x for x in nums if 条件]`
