1. 为什么 Agent 应用开发要先学 Python？

虽然前端开发者可以用 JavaScript / TypeScript 开发 AI 应用，但 Python 仍然非常重要，因为：

  1. 大量 AI / LLM / Agent 生态首先支持 Python。 - LangChain - LlamaIndex - OpenAI SDK - Anthropic SDK - MCP / 工具调用相关示例 - 数据处理、向量数据库、RAG 示例

  2. Python 更适合快速写脚本和实验。 - 读取文件 - 处理 JSON - 请求 API - 调用模型 - 组织工具函数 - 快速验证 Agent 思路

  3. 后续 Agent 应用经常需要后端能力。 你作为前端开发者，已经具备 UI、交互、组件化、状态管理经验。补上 Python 后，可以逐步具备： - 后端接口开发； - AI 服务封装； - 自动化脚本； - LLM 工具调用； - 多步骤 Agent 工作流开发。

3. Python 和 JavaScript 的核心区别

你是前端开发者，所以可以先用 JavaScript 对比 Python。

3.1 变量声明

JavaScript：

const name = "Kai"
  let age = 25

Python：

name = "Kai"
  age = 25

Python 不需要 let、const、var。

3.2 代码块

JavaScript 使用 {}：

if (age >= 18) {
    console.log("adult")
  }

Python 使用缩进：

if age >= 18:
      print("adult")

重点：Python 对缩进非常敏感。一般使用 4 个空格。

3.3 输出内容

JavaScript：

console.log("hello")

Python：

print("hello")

3.4 注释

JavaScript：

// 单行注释

Python：

# 单行注释

4. Python 基础语法

4.1 基础数据类型

name = "Alice"      # str 字符串
  age = 20            # int 整数
  height = 1.68       # float 小数
  is_student = True   # bool 布尔值
  nothing = None      # 空值

对应 JS：

const name = "Alice"
  const age = 20
  const height = 1.68
  const isStudent = true
  const nothing = null

注意：

  • Python 布尔值是 True / False，首字母大写。
  • Python 空值是 None，不是 null。

4.2 字符串

name = "Alice"
  message = "Hello, " + name
  print(message)

推荐使用 f-string：

name = "Alice"
  age = 20
  print(f"My name is {name}, I am {age} years old")

类似 JS 模板字符串：

console.log(`My name is ${name}, I am ${age} years old`)

4.3 条件判断

score = 85

  if score >= 90:
      print("优秀")
  elif score >= 60:
      print("及格")
  else:
      print("不及格")

注意：

  • Python 用 elif，不是 else if。
  • 条件后面要有冒号 :。
  • 代码块靠缩进。

4.4 循环

for 循环

names = ["Alice", "Bob", "Charlie"]

  for name in names:
      print(name)

类似 JS：

for (const name of names) {
    console.log(name)
  }

range

for i in range(5):
      print(i)

输出：

0
  1
  2
  3
  4

while 循环

count = 0

  while count < 3:
      print(count)
      count += 1

5. 常用数据结构

5.1 list 列表

类似 JS 数组。

skills = ["HTML", "CSS", "JavaScript"]

  print(skills[0])
  skills.append("Python")
  print(skills)

常用操作：

skills = ["HTML", "CSS"]

  skills.append("JavaScript")     # 追加
  skills.remove("CSS")            # 删除指定值
  print(len(skills))               # 长度
  print("HTML" in skills)         # 判断是否存在

5.2 dict 字典

类似 JS 对象。

user = {
      "name": "Alice",
      "age": 20,
      "role": "frontend developer"
  }

  print(user["name"])
  print(user.get("email"))

修改：

user["age"] = 21
  user["city"] = "Shanghai"

遍历：

for key, value in user.items():
      print(key, value)

5.3 tuple 元组

不可修改的列表。

point = (10, 20)
  print(point[0])

5.4 set 集合

用于去重和集合判断。

tags = {"python", "agent", "python"}
  print(tags)

输出只会保留一个 python。

6. 函数

JavaScript：

function add(a, b) {
    return a + b
  }

Python：

def add(a, b):
      return a + b

  result = add(1, 2)
  print(result)

带默认参数：

def greet(name="Guest"):
      print(f"Hello, {name}")

  greet()
  greet("Alice")

7. 模块和导入

Python 可以把代码拆成多个文件。

假设有 utils.py：

def format_name(name):
      return name.strip().title()

在 main.py 里使用：

from utils import format_name

  print(format_name(" alice "))

类似前端中的：

import { formatName } from "./utils"

8. JSON 处理：Agent 开发非常重要

Agent 应用中，经常会处理结构化数据，例如：

  • 模型返回的 JSON；
  • 工具调用参数；
  • API 请求结果；
  • 用户配置；
  • 对话历史。

Python 处理 JSON：

import json

  user = {
      "name": "Alice",
      "skills": ["frontend", "python"]
  }

  json_text = json.dumps(user, ensure_ascii=False, indent=2)
  print(json_text)

  parsed_user = json.loads(json_text)
  print(parsed_user["name"])

重点：

  • json.dumps()：Python 对象转 JSON 字符串。
  • json.loads()：JSON 字符串转 Python 对象。

9. 文件读写

读取文件：

with open("data.txt", "r", encoding="utf-8") as f:
      content = f.read()

  print(content)

写入文件：

with open("output.txt", "w", encoding="utf-8") as f:
      f.write("Hello Python")

读写 JSON 文件：

import json

  user = {
      "name": "Alice",
      "role": "frontend developer"
  }

  with open("user.json", "w", encoding="utf-8") as f:
      json.dump(user, f, ensure_ascii=False, indent=2)

  with open("user.json", "r", encoding="utf-8") as f:
      data = json.load(f)

  print(data["name"])
